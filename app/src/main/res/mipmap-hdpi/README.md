# App Icons

Replace the placeholder icons with your own app icons.

## Required Sizes

- **mipmap-hdpi**: 72x72 pixels
- **mipmap-mdpi**: 48x48 pixels
- **mipmap-xhdpi**: 96x96 pixels
- **mipmap-xxhdpi**: 144x144 pixels
- **mipmap-xxxhdpi**: 192x192 pixels

## Files to Replace

In each mipmap folder, replace:
- `ic_launcher.png` - App icon (square)
- `ic_launcher_round.png` - Round app icon (for devices that support it)

## Online Icon Generators

Use these tools to generate all required sizes:
- https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
- https://appicon.co/
- https://icon.kitchen/

## Current Status

The current icons are Android's default launcher icons. They will work but you should replace them with your own branding.
